# Image Background Remover

**A Python tool to automatically remove backgrounds from images with ease!**

[![Python](https://img.shields.io/badge/Python-3.13-blue)](https://www.python.org/downloads/release/python-3132/)
[![License: MIT](https://img.shields.io/badge/License-MIT-green)](LICENSE)

Remove backgrounds from images effortlessly using AI-powered background removal. Perfect for product photography, profile pictures, or creative projects!
## Run Locally

Clone the project

```bash
  git clone https://github.com/therishabhbaghel/RemoveBG
```

Go to the project directory

```bash
  cd RemoveBG
```

Install dependencies

```bash
  pip install -r requirements.txt
```

Start the server

```bash
  python main.py
```


## Features

- Supports multiple formats
- Fast Processing
- Easy to use
